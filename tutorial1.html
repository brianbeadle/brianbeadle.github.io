Introduction to Beamer presentations using LaTeX

Introduction

In this tutorial, I am going to explain how to build a basic presentation in LaTeX. We will go over the commands to specify the document class, load packages, and build slides containing text, hyperlinks, columns, images, and basic tables.

Preamble commands

To begin, we will specify the beamer document class that is designed for building slides:


   \documentclass{beamer}
  Copy

Next, we will want to choose a theme. There are a plethora of themes to choose from, and I recommend taking a few moments to review the theme gallery provided by UAB Barcelona.

I personally like the Berlin theme because of the progress bar with clickable dots for each slide in the navigation bar, but you are welcome to choose any theme you find interesting.

In addition to the Berlin theme, I add commands to give the appearance a bit more flavor. The option compress makes the navigation bar as small as possible, and I use the crane color theme and the structuresmallcapsserif font option. 


   \usetheme[compress]{Berlin}
   \usecolortheme{crane}                     % specifies the colors of the background and font
   \usefonttheme{structuresmallcapsserif}    % specifies the font style
  Copy

Finally, we need to specify a few packages to make sure we have the necessary options for our slides.


   \usepackage{hyperref}                     % gives us the option to use hyperlinks
  Copy

Finally, we use the command to end the preamble and begin the visible portion of the document:


   \begin{document}              
  Copy

Document

Now we can begin creating slides (i.e. frames)! Let's begin with a basic slide that has a title and text in the body. We can also specify the introductory section of the presentation, which will appear as a clickable title in the navigation bar of the Berlin theme we are using.


   \section{Introduction}            % creates the section for the navigation bar
   \begin{frame}{Cover slide}        % begins the first slide with the title "Cover slide"
      My Beamer presentation         % add your content here
      Author: You
      Date: \today                   % the \today command automatically sets the date
    \end{frame}                      % ends the slide
